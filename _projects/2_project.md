[ctpp-mcp](https://github.com/hesscl/ctpp-mcp) is a Model Context Protocol (MCP) server written in TypeScript that provides AI assistants with access to the Census Transportation Planning Products (CTPP) database — a specialized dataset on commuting patterns, workplace characteristics, and transportation mode use derived from the American Community Survey.

**Features:**
- List and search CTPP datasets and table groups
- Retrieve variable definitions for any CTPP table
- Fetch aggregate data for states, counties, tracts, and transportation analysis zones
- Resolve geography FIPS codes for filtering

**Use cases:** Analyzing commute flows between home and work locations, studying transportation mode disparities across demographic groups, supporting urban planning research.
