[wonder-mcp](https://github.com/hesscl/wonder-mcp) is a Model Context Protocol (MCP) server written in Python that connects AI assistants to CDC WONDER — the Centers for Disease Control's public health data query system covering mortality, natality, cancer incidence, infectious disease, environmental health, and more.

**Features:**
- Query mortality and cause-of-death statistics by year, geography, age, race, and sex
- Access natality (birth) data with demographic breakdowns
- Retrieve cancer incidence and survival data
- Support for geographic aggregation from national to county level

**Use cases:** Epidemiological research, public health surveillance, studying health disparities across demographic groups and geographies.
